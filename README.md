# Kindle-Gallery: 📚 我的 Kindle 策展合集

> 「选择屏保图片时，就像在给自己策划一场私人展览。」

这个仓库收集了一系列精选的电影海报、展览海报、书籍封面及极简设计作品。它们不仅是 Kindle 的屏保，更是你阅读时光中的视觉点缀，让你的电子阅读器在闲置时也能成为一个有质感的艺术摆件。

## 🛠️ 如何使用 (How to Use)

### 对于已越狱并安装 KOReader 的 Kindle：
1.  **下载：** 克隆本仓库或下载你喜欢的 ISSUE 文件夹。
2.  **传输：** 将图片通过 USB 连接传送到 Kindle。建议存放路径：`koreader/screensaver/`。
3.  **设置：** 在 KOReader 中进入 `设置 -> 屏幕 -> 屏保`，选择该文件夹作为屏保来源。
4.  **效果：** 每次关机/待机时，KOReader 会随机展示该文件夹中的艺术作品。

### 对于普通 Kindle (未越狱)：
*   如果你正在阅读一本书，Kindle (支持显示封面的版本) 可以显示当前书籍封面。
*   你可以利用一些工具（如 Send-to-Kindle）将图片作为“文档”发送，虽然无法直接设为系统屏保，但可以作为电子画册浏览。

## 🎨 建议与贡献
*   **分辨率：** 本仓库图片主要适配 Kindle Paperwhite 及更高机型，建议分辨率为 `1072x1448` 或更高。
*   **色彩：** 针对 E-ink 屏幕特性，建议使用对比度较高的黑白或低饱和度图片以获得最佳显示效果。
*   **更新：** 策展持续进行中，欢迎关注 **ISSUE #05** 及后续更新。
